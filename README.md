# CSC365 Group Project  
**Pitch:**  
Our project will be a backend API for a library system. This will involve keeping a catalog of the items the library offers (books, dvds, etc) and keeping a record of the items being checked out and returned by patrons. This also involves keeping track of accounts for library patrons, and account creation. Users will be able to leave ratings of the books they have checked out.
