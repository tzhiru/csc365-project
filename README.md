# CSC365 Group Project  
**Pitch:**  
Our project will be a backend API for a library system. This will involve keeping a catalog of the items the library offers (books, dvds, etc) and keeping a record of the items being checked out and returned. This also involves keeping track of accounts for library patrons, and account creation.
